Fields:

Click on one upper-case entry if you want to load an image. Multiple upper-case entries will (hopefully) generate a montage or blinking animation.

Click on one or more lower-case entries to overlay contour plots.

On some machines (e.g. PCs with windows) you may need to hold down the CTRL key while clicking for multiple choices with the mouse.

Field Definitions:

HT: height
TEMP: temperature
SFCT: surface temperature
DEWPOINT: dewpoint
RH: relative humidity
SH: specific humidity
MR: mixing ratio
WIND_SPD: wind speed
WIND_U: wind u component
WIND_V: wind v component
WIND_DIV: divergence
WIND_VORT: vorticity
WIND_PVORT: potential vorticity
WIND_OMEGA: omega (vertical velocity)
C_OMEGA: cloud omega
MCONV: moisture flux converence
P_MSL: mean sea level pressure
P_PRIME: perturbation pressure
THETA: potential temperature
THETAE: equivalent potential temperature
WTBLB0: height of wet bulb zero temperature
CAPE: convective available potential energy
CIN: convective inhibition
CLD_ALB: cloud albedo
CLD_ALB1: cloud albedo (LCV/CLA)
CLD_BASE: cloud base height
CLD_CEIL: cloud ceiling height
CLD_CVR: cloud fraction
CLD_TOP: cloud top height
CLD_LIQ: cloud liquid
CLD_ICE: cloud ice
HELICITY: storm relative helicity
UMFLUX: upslope moisture flux
VSBLTY: visibility
HEATINDEX: heat index
FIREWX: fire weather/danger index
FIRE_FBG: Fosberg fire weather index
VENT_INDX: ventilation index
HAINES: Haines fire weather index
SAT_11U: 11 micron IR brightness temperature
SAT_12U: 12 micron IR brightness temperature
SAT_WV: satellite water vapor channel
SAT_39U: satellite 3.9 micron brightness temperature
SAT_D39: 3.9 and 11 micron brightness temperature difference
SAT_VIS: inactive - possibly use (LVD/SVS) for reflectance?
SAT_VIS_ALB: satellite albedo derived from reflectance (LCV/ALB)
REFLCT: radar reflectivity
REF_MAX: column maximum radar reflectivity
PCP_RAI: rain concentration
PCP_SNO: snow concentration
PCP_ICE: ice concentration
PCP_INC: incremental precip
PCP_STO: storm total precip
PRCP_H2O: precipitable water
SNOW_1HR: incremental snow accumulation
SNOW_STO: storm total snow accumulation
SNOW_CVR: fractional snow cover
LAND_FRAC: land/water fraction
LE_FLUX: latent heat flux
SE_FLUX: sensible heat flux
LONG_WAVE: outgoing long wave radiation
SW_DOWN: downward short wave radiation
SW_OUT: outgoing short wave radiation
GG: various static fields (e.g. alb,lu using "Initial Time" window)
radial_vel: radial velocity (red is outbound, green is inbound, enter radar # using "Initial Time" window, 0 is all radars)
